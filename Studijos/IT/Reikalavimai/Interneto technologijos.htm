  2. XML Schema (1 balas, atsiskaityti reikia iki balandþio 18 d. imtinai) Reikia paraðyti XML Schema dokumentà, apibrëþiantá jûsø sukurto XML dokumento struktûrà, kuriame bûtø jûsø paèiø susikurti:
    1. Paprastas tipas, apribojantis koká nors Schema standartiná tipà,
    2. Prasmingas sàraðo tipas (list type). Neprasmingo sàraðo tipo pavyzdys: sàraðas ið string tipo elementø. Neprasmingas todël, kad já galima pakeisti vienu paprastu tipu: string.
    3. Sudëtingi tipai su paprastu turiniu,
      1. iðvesti apribojimu (restriction) ið jûsø (ne XML Schema) bazinio tipo
      2. iðvesti praplëtimu (extension) ið jûsø bazinio tipo
    4. Sudëtingi tipai su sudëtingu turiniu,
      1. iðvesti apribojimu (restriction) ið jûsø bazinio tipo
      2. iðvesti praplëtimu (extension) ið jûsø bazinio tipo
    5. Sudëtingas tipas su miðriu turiniu,
    6. choice valdymo struktûra,
    7. Bent viena nuoroda (keyref) á unikalumo ribojimà (unique) arba raktà (key)
    8. Visi jûsø susikurti tipai turi priklausyti jûsø vardø srièiai
