On slower computers it may take some time until the image appears. At first, the browser must load its Java environment, then the browser must read the JavaView archive from disk, and finally the browser must run JavaView viewer.

To display your own geometries just replace the name "sample.obj" in the HTML code of this page with the name of your geometry file. Possible file formats and further details on including applets are discussed in the JavaView tutorial.
